# Hybrid Contracts

Standalone package that provides general-usage interfaces for the Hybrid Core framework.

## Requirements

* WordPress 4.9.6+.
* PHP 7.4+ (8.0+ recommended).
* [Composer](https://getcomposer.org/) for managing PHP dependencies.
